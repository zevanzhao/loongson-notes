# 福珑2中设置串口登录
1. 在BIOS中开启串口重定向。
2. 在grub.cfg中，打开串口，命令行参数上加上
```
root=/dev/sda6 ro rhgb quiet loglevel=0 LANG=zh_CN.UTF-8 console=ttyS0,115200  console=tty
```
3. 使用USB串口线，需要将Minicom的串口设备设置为/dev/ttyUSB0
4. 解决minicom无法输入的问题，设置minicom，调整键盘输入：
将F - Hardware Flow Contorl项[从Yes，修改为No](https://blog.csdn.net/mirkerson/article/details/7377048)

如果有必要，在系统中[允许串口的登录](https://blog.csdn.net/zzwdkxx/article/details/91374032)
修改/lib/systemd/system/serial-getty@.service
如果设置为root自动登录，这样修改：
```
[Service]
Environment="TERM=xterm"
ExecStart=-/sbin/agetty -a root -8 -L %I 115200 $TERM
```
